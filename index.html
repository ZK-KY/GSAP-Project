Deen Developers Hackathon Submission: Unseen

For the Deen Developers Hackathon, we built Unseen, a secure, privacy-first web platform that connects individuals in need of legal assistance with vetted legal professionals. Using Next.js, MySQL, and Google Cloud, we focused on making legal support more accessible and responsive, with features like case submissions, filtering, and secure data handling

Ray Casted Capture the Flag Game

For my A-Level Computer Science NEA, I built a Ray-Casted Capture the Flag game in Python using Tkinter and Matplotlib. The simulation features AI agents navigating a maze with Dijkstra's Algorithm and ray casting for vision.

Modelling 3D Projectile Motion

Vuilt an interactive simulation in **Python** using Tkinter and Matplotlib to visualise the trajectory of objects in three-dimensional space. The tool allows users to input initial velocity, angle, and height parameters, and observe real-time plots of motion, including position, range, and time of flight.

Local Voice Activated LLM Chatbot

Developed an offline conversational assistant that uses Google Text-to-Speech for vocal responses and Ollama to run a locally hosted LLM. This project explored local AI deployment, aiming to make conversational AI more secure and usable in low-connectivity environments.
